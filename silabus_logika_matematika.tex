\section*{2. Capaian Pembelajaran Lulusan (CPL)}

CPL yang dibebankan pada mata kuliah ini mencakup kompetensi lulusan dalam aspek pengetahuan, keterampilan, dan sikap:

\begin{itemize}[leftmargin=*]
  \item \textbf{CPL-1 (Pengetahuan):} Menguasai konsep teoretis logika matematika dan fondasi diskrit yang menjadi dasar ilmu komputer serta mampu memformulasikan argumen dan bukti secara formal.
  
  \item \textbf{CPL-2 (Keterampilan Umum):} Mampu menerapkan pemikiran logis, kritis, sistematis, dan analitis dalam konteks pemecahan masalah informatika serta pengembangan atau implementasi ilmu pengetahuan dan teknologi.
  
  \item \textbf{CPL-3 (Keterampilan Khusus):} Mampu menganalisis, memodelkan, dan memverifikasi pernyataan menggunakan logika proposisional, logika predikat, serta mampu menggunakan konsep logic programming untuk pemecahan masalah.
  
  \item \textbf{CPL-4 (Sikap):} Menunjukkan sikap bertanggung jawab atas pekerjaan di bidang keahliannya secara mandiri, teliti dalam penalaran, dan mampu bekerja sama dalam tim.
\end{itemize}

\vspace{0.5cm}

% ============================================================
% 3. CAPAIAN PEMBELAJARAN MATA KULIAH (CPMK)
% ============================================================
\section*{3. Capaian Pembelajaran Mata Kuliah (CPMK)}

Kemampuan atau kompetensi spesifik yang diharapkan mahasiswa kuasai setelah menyelesaikan mata kuliah:

\begin{itemize}[leftmargin=*]
  \item \textbf{CPMK-1:} Mahasiswa mampu memahami dan menjelaskan konsep proposisi, operator logika (negasi, konjungsi, disjungsi, implikasi, bikondisional), serta membangun dan menganalisis tabel kebenaran.
  
  \item \textbf{CPMK-2:} Mahasiswa mampu menerjemahkan pernyataan bahasa alami ke bentuk logika formal dan menganalisis ekuivalensi logika serta tautologi, kontradiksi, dan kontingensi.
  
  \item \textbf{CPMK-3:} Mahasiswa mampu memahami logika predikat (kuantifier universal dan eksistensial), mengevaluasi kebenaran pernyataan berkuantifier, dan menerapkan aturan inferensi untuk membangun bukti formal.
  
  \item \textbf{CPMK-4:} Mahasiswa mampu menggunakan berbagai metode bukti (langsung, kontraposisi, kontradiksi, induksi matematika) dan mengenal dasar-dasar logic programming dengan Prolog.
\end{itemize}

\vspace{0.5cm}

% ============================================================
% 4. SUB-CPMK / INDIKATOR PENCAPAIAN
% ============================================================
\section*{4. Sub-CPMK / Indikator Pencapaian}

Penjabaran CPMK menjadi indikator yang lebih terukur dan dapat diuji:

\begin{itemize}[leftmargin=*]
  \item \textbf{Sub-CPMK 1.1:} Menjelaskan definisi proposisi dan membedakan proposisi dari kalimat non-proposisi
  \item \textbf{Sub-CPMK 1.2:} Menggunakan operator logika (not, and, or, implies, iff) untuk membentuk proposisi majemuk
  \item \textbf{Sub-CPMK 1.3:} Membuat tabel kebenaran untuk proposisi majemuk dan menentukan nilai kebenarannya
  \item \textbf{Sub-CPMK 2.1:} Menerjemahkan pernyataan bahasa Indonesia ke simbol logika proposisional
  \item \textbf{Sub-CPMK 2.2:} Menentukan ekuivalensi logika menggunakan hukum-hukum aljabar proposisi
  \item \textbf{Sub-CPMK 2.3:} Mengidentifikasi tautologi, kontradiksi, dan kontingensi
  \item \textbf{Sub-CPMK 3.1:} Menulis dan mengevaluasi pernyataan berkuantifier universal ($\forall$) dan eksistensial ($\exists$)
  \item \textbf{Sub-CPMK 3.2:} Menggunakan aturan inferensi (modus ponens, modus tollens, silogisme, dll.) untuk membangun argumen valid
  \item \textbf{Sub-CPMK 4.1:} Membuktikan pernyataan menggunakan metode bukti langsung dan kontraposisi
  \item \textbf{Sub-CPMK 4.2:} Membuktikan pernyataan menggunakan induksi matematika
  \item \textbf{Sub-CPMK 4.3:} Menulis fakta dan aturan sederhana dalam Prolog serta menjalankan query dasar
\end{itemize}

% ============================================================
% 6. METODE PEMBELAJARAN
% ============================================================
\section*{6. Metode Pembelajaran}

Strategi atau pendekatan pembelajaran yang dipilih sesuai OBE yang menekankan aktivitas mahasiswa:

\begin{itemize}[leftmargin=*]
  \item \textbf{Ceramah Interaktif:} Penjelasan konsep logika dengan diskusi tanya jawab dan contoh-contoh relevan untuk informatika
  \item \textbf{Problem-Based Learning (PBL):} Mahasiswa menyelesaikan permasalahan logika (translasi, tabel kebenaran, bukti) secara mandiri atau berkelompok
  \item \textbf{Latifan Terstruktur:} Pengerjaan soal-soal logika dan bukti di kelas dengan bimbingan dosen
  \item \textbf{Demonstrasi:} Penggunaan tools online (ProofLab, TruthLab, Carnap) untuk latihan natural deduction dan tabel kebenaran
  \item \textbf{Praktikum Prolog:} Pengenalan SWI-Prolog dan latihan penulisan fakta, aturan, serta query
  \item \textbf{Flipped Classroom:} Mahasiswa mempelajari materi sebelum kelas melalui OER, diskusi mendalam di kelas
  \item \textbf{Studi Kasus:} Analisis penerapan logika dalam sirkuit digital, verifikasi perangkat lunak, atau kecerdasan buatan
\end{itemize}

% 7. PENGALAMAN BELAJAR MAHASISWA

\section*{7. Pengalaman Belajar Mahasiswa}

Deskripsi tugas, aktivitas, atau pengalaman belajar yang mendukung pencapaian Sub-CPMK:

\begin{itemize}[leftmargin=*]
  \item Menerjemahkan pernyataan sehari-hari atau spesifikasi sederhana ke bentuk logika proposisional/predikat
  \item Membuat tabel kebenaran untuk proposisi majemuk dan menganalisis hasilnya
  \item Membuktikan ekuivalensi logika menggunakan hukum-hukum aljabar proposisi
  \item Membangun bukti formal menggunakan aturan inferensi (modus ponens, modus tollens, silogisme, dll.)
  \item Membuktikan pernyataan numerik atau kombinatorial menggunakan induksi matematika
  \item Menggunakan tools online (proof checker, truth table generator) untuk memverifikasi jawaban
  \item Menulis program Prolog sederhana untuk memodelkan relasi (misal: silsilah keluarga, aturan bisnis)
  \item Berkolaborasi dalam tim kecil untuk menyelesaikan soal bukti yang lebih kompleks
  \item Merefleksikan kaitan logika matematika dengan mata kuliah lain (struktur data, basis data, kecerdasan buatan)
\end{itemize}
