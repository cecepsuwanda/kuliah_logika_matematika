# Prompt Pencarian Referensi Sumber Terbuka Logika Matematika

**Instruksi:** 
Silakan salin dan tempel (copy-paste) prompt di bawah ini ke dalam AI interaktif (seperti ChatGPT, Claude, atau Gemini) untuk menjalankan tugas pencarian otomatis.

***

**Prompt:**

```text
Tolong bertindak sebagai asisten akademik dan asisten peneliti tingkat lanjut. Tugas Anda adalah mencari, mengumpulkan, dan merangkum referensi sumber terbuka (open-source) di internet mengenai materi **Logika Matematika** tingkat perguruan tinggi.

Ketentuan dan metrik pencarian yang harus Anda penuhi:
1. Cari dokumen **Rencana Pembelajaran Semester (RPS)** untuk Matakuliah Logika Matematika (atau ekuivalen) dari berbagai perguruan tinggi (minimum 5 referensi). Untuk setiap RPS, cantumkan nama institusi, program studi, dan topik-topik utama yang diajarkan, serta tautan sumbernya jika memungkinkan.
2. Cari materi, bahan ajar, atau referensi terkait **Matakuliah Logika Matematika** (minimum 50 referensi). Referensi ini harus berasal dari sumber daya terbuka (OER). Anda dapat menggabungkannya dari berbagai kelompok berikut:
   - Buku teks bersumber terbuka (Open Textbooks / e-Books) berbahasa Inggris maupun Indonesia.
   - Modul kuliah, diktat, atau bahan ajar dari universitas-universitas terbuka/digital.
   - Kursus terbuka (seperti MIT OpenCourseWare, Coursera, NPTEL, dsb).
   - Jurnal ilmiah pendidikan matematika dan logika.
   - Video pembelajaran atau repositori GitHub yang relevan dengan pembuktian/logika matematis.
3. Kelompokkan ke-50 referensi materi tersebut ke dalam beberapa kategori/topik agar rapi dan mudah dibaca (misalnya: Buku Teks, Modul Universitas, Kursus Terbuka, Artikel Jurnal, dan Sumber Multimedia).
4. Hasil akhir harap ditulis dan diformat menggunakan Markdown secara lengkap, menyertakan ke-5 RPS dan ke-50 Referensi Materi tersebut, agar output-nya siap digunakan sebagai file `sumber_materi.md`.
```
