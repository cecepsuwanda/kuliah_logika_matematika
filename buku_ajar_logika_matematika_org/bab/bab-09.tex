\begin{aktivitas}
  \item Verifikasi Postulat Huntington: Buktikan hukum idempoten ($x + x = x$) secara aljabar menggunakan Postulat Huntington (ketertutupan, identitas, komplemen, distributif). Tuliskan setiap langkah beserta justifikasi postulatnya.
  \item Evaluasi fungsi Boolean: Diberikan $F(x,y,z) = x'y + yz' + x'z$, buatlah tabel kebenaran lengkap ($2^3 = 8$ baris) dengan kolom antara untuk setiap sub-ekspresi.
  \item Penyederhanaan aljabar: Sederhanakan ekspresi $G = x'y'z + x'yz + xy'z + xyz$ menggunakan hukum-hukum aljabar Boolean. Tuliskan setiap langkah dan hukum yang digunakan.
  \item Konversi SOP $\leftrightarrow$ POS: Ubah ekspresi $F(x,y,z) = \sum m(0, 2, 5, 7)$ ke bentuk kanonik POS ($\prod M$). Verifikasi kebenaran konversi Anda.
  \item Buktikan hukum konsensus $xy + x'z + yz = xy + x'z$ menggunakan hukum-hukum aljabar Boolean. Jelaskan secara intuitif mengapa suku $yz$ bersifat redundan.
  \item Implementasi gerbang: Gambarkan rangkaian gerbang logika untuk ekspresi $H = (A + B)(A' + C)$, kemudian sederhanakan dan gambarkan ulang rangkaian yang telah disederhanakan. Bandingkan jumlah gerbang sebelum dan sesudah penyederhanaan.
  \item Desain rangkaian: Rancang rangkaian voting mayoritas untuk 3 pemilih menggunakan hanya gerbang NAND. Tunjukkan bahwa fungsi $F = AB + AC + BC$ dapat diimplementasikan seluruhnya dengan gerbang NAND.
\end{aktivitas}

\begin{latihan}
  \item Sederhanakan fungsi Boolean $F(x, y, z) = x'y'z + x'yz + xy'$ menggunakan hukum-hukum aljabar Boolean. Tuliskan justifikasi hukum pada setiap langkah.
  \item Tuliskan ekspresi Minterm ($\sum m$) dan Maxterm ($\prod M$) untuk fungsi $F(x, y, z) = xy + z'$. Verifikasi bahwa kedua representasi menghasilkan tabel kebenaran yang identik.
  \item Diberikan fungsi $G(A, B, C) = \sum m(1, 2, 3, 5, 7)$.
  \begin{enumerate}
    \item Tuliskan bentuk kanonik SOP lengkap.
    \item Konversikan ke bentuk kanonik POS ($\prod M$).
    \item Sederhanakan bentuk SOP menggunakan aljabar Boolean.
  \end{enumerate}
  \item Buktikan dengan aljabar Boolean bahwa $(x + y)(x' + z)(y + z) = (x + y)(x' + z)$ (hukum konsensus versi POS).
  \item Tentukan apakah himpunan gerbang $\{NOR\}$ bersifat lengkap secara fungsional. Tunjukkan bagaimana mengimplementasikan NOT, AND, dan OR hanya menggunakan gerbang NOR.
  \item Rancang sebuah fungsi Boolean untuk sistem kontrol lampu lalu lintas sederhana dengan 2 sensor ($S_1$, $S_2$) dan output lampu hijau ($G$). Lampu hijau menyala jika minimal satu sensor mendeteksi kendaraan. Tuliskan ekspresi Boolean, sederhanakan, dan gambarkan rangkaian gerbangnya.
  \item \textbf{Refleksi}: Bagaimana Anda melihat kemiripan antara aljabar Boolean dengan teori himpunan dan logika proposisi? Apakah pemahaman terhadap satu bidang membantu memahami bidang lainnya? Jelaskan dengan contoh konkret dari hukum De Morgan pada ketiga sistem.
  \item \textbf{Refleksi}: Dalam konteks pemrograman, kapan Anda menggunakan penyederhanaan ekspresi Boolean? Berikan contoh kondisi \texttt{if} yang kompleks dalam Python yang dapat disederhanakan menggunakan hukum-hukum aljabar Boolean.
\end{latihan}

\textbf{Instrumen Evaluasi Kinerja untuk Sub-CPMK 3.1--3.3}

\textbf{A. Pilihan Ganda}
\begin{enumerate}
  \item Hukum De Morgan untuk fungsi Boolean $(AB)' $ adalah:
  \begin{enumerate}
    \item $A'B'$
    \item $A' + B'$
    \item $(A+B)'$
    \item $A+B$
  \end{enumerate}
  Jawaban: (b)

  \item Yang \emph{bukan} merupakan Postulat Huntington adalah:
  \begin{enumerate}
    \item Ketertutupan
    \item Komutatif
    \item Asosiatif
    \item Distributif
  \end{enumerate}
  Jawaban: (c) --- Hukum asosiatif adalah teorema turunan, bukan postulat.

  \item Minterm $m_5$ untuk 3 variabel $(x, y, z)$ adalah:
  \begin{enumerate}
    \item $x'y'z$
    \item $xy'z$
    \item $xyz'$
    \item $x'yz$
  \end{enumerate}
  Jawaban: (b) --- $5 = 101_2$, sehingga $x = 1, y = 0, z = 1 \Rightarrow xy'z$.

  \item Gerbang yang bersifat lengkap secara fungsional (\textit{functionally complete}) secara individual adalah:
  \begin{enumerate}
    \item AND
    \item OR
    \item NAND
    \item XOR
  \end{enumerate}
  Jawaban: (c)

  \item Dual dari ekspresi $x + (y \cdot z) = (x + y) \cdot (x + z)$ adalah:
  \begin{enumerate}
    \item $x \cdot (y + z) = (x \cdot y) + (x \cdot z)$
    \item $x' + (y' \cdot z') = (x' + y') \cdot (x' + z')$
    \item $x + (y + z) = (x + y) + z$
    \item $(x \cdot y)' = x' + y'$
  \end{enumerate}
  Jawaban: (a)
\end{enumerate}

\begin{rangkuman}
Bab ini mengkaji Aljabar Boolean sebagai sistem aljabar fundamental yang beroperasi pada himpunan $B = \{0, 1\}$ dengan tiga operator dasar: penjumlahan Boolean (OR, $+$), perkalian Boolean (AND, $\cdot$), dan komplemen (NOT, $'$).

\textbf{Poin Kunci:}
\begin{itemize}
  \item Aljabar Boolean diperkenalkan oleh George Boole (1854) dan diterapkan pada rangkaian digital oleh Claude Shannon (1938). Sistem ini bersifat isomorfik dengan logika proposisi dan teori himpunan, sehingga hukum-hukum pada satu sistem dapat diterjemahkan ke sistem lainnya.
  \item Postulat Huntington (1904) mendefinisikan aljabar Boolean melalui enam aksioma: ketertutupan, identitas, komutatif, distributif, komplemen, dan eksistensi minimal. Seluruh hukum Boolean lainnya (termasuk asosiatif, idempoten, absorpsi, De Morgan, dan konsensus) merupakan teorema yang dapat diturunkan dari postulat-postulat ini.
  \item Prinsip dualitas menyatakan bahwa setiap identitas Boolean tetap valid jika operator $+$ dan $\cdot$ ditukar, serta konstanta $0$ dan $1$ ditukar secara simultan.
  \item Penyederhanaan ekspresi Boolean menggunakan hukum-hukum aljabar bertujuan meminimalkan jumlah literal dan operator, yang berimplikasi langsung pada pengurangan jumlah gerbang logika dalam implementasi rangkaian digital.
  \item Fungsi Boolean dapat direpresentasikan dalam bentuk kanonik Sum of Products (SOP, menggunakan minterm $\sum m$) atau Product of Sums (POS, menggunakan maxterm $\prod M$), yang keduanya memberikan representasi unik dan dapat dikonversi satu sama lain.
  \item Gerbang logika (AND, OR, NOT, NAND, NOR, XOR) merupakan implementasi fisik dari operator Boolean. Gerbang NAND dan NOR masing-masing bersifat lengkap secara fungsional (\textit{functionally complete}).
\end{itemize}

\textbf{Kata Kunci}: aljabar Boolean, George Boole, Claude Shannon, Postulat Huntington, prinsip dualitas, penyederhanaan aljabar, hukum konsensus, literal, minterm, maxterm, Sum of Products (SOP), Product of Sums (POS), gerbang logika, kelengkapan fungsional, NAND, NOR
\end{rangkuman}
