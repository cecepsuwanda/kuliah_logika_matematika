\chapter{Landasan Teori dan Konsep Dasar Logika Matematika}

Setelah menamatkan bab ini, mahasiswa komputer dilarang keras berasumsi bahwa logika matematika sebatas hafalan kaku peninggalan filsuf masa lampau. Mahasiswa diwajibkan menyibak dimensi epik bagaimana aljabar purba menjelma menjadi fondasi utama \textit{coding software}, mesin kecerdasan buatan, maupun miliaran sirkuit kepingan \textit{Integrated Circuits} di ranah piranti keras masa depan.

\begin{subcpmk}
  \item Sub-CPMK 1.1: Mahasiswa mampu menjelaskan konsep dasar proposisi, nilai kebenaran, dan kata hubung logika (konjungsi, disjungsi, negasi, implikasi, biimplikasi).
  \item Sub-CPMK 1.2: Mahasiswa mampu menggunakan tabel kebenaran untuk mengevaluasi pernyataan majemuk serta mengidentifikasi tautologi, kontradiksi, dan kontingensi.
  \item Sub-CPMK 1.3: Mahasiswa mampu menganalisis validitas argumen menggunakan aturan inferensi (modus ponens, modus tollens, silogisme) dalam memecahkan persoalan komputasi dasar.
\end{subcpmk}

\begin{asesmen}
\textbf{Instrumen Penilaian untuk Sub-CPMK Bab 2}

\textbf{A. Pemilihan Ganda Kritis}

\begin{enumerate}
  \item Ketika George Boole mengejutkan sains di tahun 1854 lewat *The Laws of Thought*, terobosan radikal apa yang murni ia wariskan sebagai "otak" bagi fondasi mikroprosesor transistor CPU di seabad kemudiannya?
  \begin{enumerate}
    \item Merancang gerbang listrik 5 Volt.
    \item Mendikte hukum inferensi Modus Ponens bahasa awam.
    \item Merebus alam perdebatan retorika utuh menjadi matematika kaku kalkulasi operasi Biner **(1/0)** Boolean.
    \item Menemukan cip arsitektur motherboard tembaga.
  \end{enumerate}
  Jawaban: (c)

  \item Argumen: "Jika Naga itu binatang menyusui mamalia, maka sang Naga betina bertelur di dasar samudera." Kebetulan Naga adalah amfibi menyusui (T/benar berdasar cerita fiksi), maka kesimpulan sah "Naga betina niscaya bertelur di dasar samudra!". Status penilaian Validitas argumen ini di pengadilan Aljabar Inferensi adalah...
  \begin{enumerate}
    \item Cacat Invalid, karena naga itu binatang dongeng pembohong.
    \item Valid Mutlak Secara Struktur, lantaran ia murni taat asas panah inferensi tanpa peduli absurdnya biologi dunia riil premis.
    \item Tidak dapat dipastikan takdirnya gara-gara variabel Naga belum dimasukkan ke himpunan.
    \item Sesat pikir penegasan berbalik.
  \end{enumerate}
  Jawaban: (b)

  \item Seorang *Programmer Game* mengetik percabangan pelindung tameng Karakter: `IF (Status\_Dewa OR Armor\_Emas) AND (NOT Terkena\_Racun)`. Jika pemain terlanjur dihujani cairan beracun pekat (True) tetapi sombong memakai sepasang Armor Emas lengkap (True), nyawa karakter ini di mata Mesin Biner...
  \begin{enumerate}
    \item Berhasil selamat merajai game, lewat kompromi OR.
    \item Mutlak gugur tertolak mati; karena syarat tameng lapis *AND NOT Ke-Racun* jebol runtuh menabrak batu karang (*False*).
    \item Kebal oleh karena \textit{Status Dewa} menang.
    \item Tumbang diam tak berdosa.
  \end{enumerate}
  Jawaban: (b)
\end{enumerate}

\begin{rangkuman}
Bab 2 menumpas habis segala miskonsepsi awal filosofi Logika Komando.

\textbf{Pilar Kristalisasi Pemahaman Epik:}
\begin{itemize}
  \item Logika bermula membedah bahasa awam (*Syllogistic*) dari filsuf Aristoteles, membatu kaku menjadi operasi matematika biner seutuhnya ditangan Dewa Simbolik \textbf{George Boole}.
  \item \textbf{Segregasi Evaluasi Validitas} membedakan ketat antara "kebenaran faktual semesta alam dunia nyata" DENGAN pakem keabsahan mesin mutlak struktur *Inferensi Argumentative* (Biarpun premis membual halusinasi melahirkan Naga beranak macan, asal mematuhi tata letak alur mesin inferensi pelatuk, \textbf{Sah ia divonis VALID}! Secara koridor mesin formal).
  \item Mesin *Compiler Software* modern menghisap mutlak sari-sari **Kalkulus Boolean Algoritma Control Flow** menghidupi denyut percabangan IF-ELSE rumit pencetak algoritma IT dunia.
  \item Logika Predikat (Kuantifier Raksasa) menghela kemudi pencarian seribu serentak komposit baris matriks Data Tabel menelurkan rahim **Aljabar SQL Basis Data** dan \textit{Inference Rule Mesin Diagnosis Medis AI}.
  \item Perpaduan fisika Voltan Tegangan Listrik High (1)/Low (0) sukses menerjemahkan hukum abstrak menitis bodi raga **Gerbang Logika Sirkuit Silicon (AND, OR, NOT Gate Hardware)**. Memangkas barisan Boolean di angkasa mutlak mensabotase kucuran milyaran budget produksi kabel tembaga cetak semikonduktor fabrikasi riil CPU!
\end{itemize}
